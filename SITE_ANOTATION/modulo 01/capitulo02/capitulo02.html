Aula 1 - Como a internet funciona?

Represantação de dados

    A representação de dados no computador é feita por meio de sistemas binários, onde as informações são codificadas usando apenas dois símbolos: 0 e 1, chamados de bits. Cada bit representa a menor unidade de informação. Esses bits podem ser agrupados em bytes (8 bits), que são usados para representar dados mais complexos, como caracteres, números e imagens. Dependendo do tipo de dado, a representação varia:


  • Texto:

    • Usando códigos como ASCII ou Unicode, onde cada caractere é associado a um número.


  • Números:

    • Representados em formatos binários, como inteiros ou números de ponto flutuante (com a ajuda de uma representação em complemento de dois ou IEEE 754, por exemplo).


  • Imagens e áudio:

    • Representados por meio de arquivos digitais que armazenam dados binários que, quando interpretados, formam as informações visuais ou sonoras. Essa representação binária é fundamental para que o computador processe, armazene e transmita informações.



UTF-8

    O UTF-8 é uma codificação de caracteres amplamente utilizada na informática para representar texto. Ele é parte do padrão Unicode e é capaz de representar todos os caracteres de praticamente todas as línguas do mundo.


  • A principal característica do UTF-8:

    • É ser uma codificação variável: usa de 1 a 4 bytes para representar um caractere. Caracteres mais comuns, como os do alfabeto latino, são representados com 1 byte, enquanto caracteres menos comuns, como ideogramas ou emojis, podem usar até 4 bytes.


    Além de ser eficiente em termos de espaço para idiomas que usam caracteres simples, como o inglês, o UTF-8 também é compatível com sistemas legados que utilizam ASCII, já que o UTF-8 usa a mesma representação para os primeiros 128 caracteres do ASCII. Ele é o formato de codificação mais utilizado na web devido à sua flexibilidade e compatibilidade.


    As ondas quadradas são usadas para representar bits em sistemas digitais, onde o nível alto (1) e o nível baixo (0) da onda alternam-se rapidamente. Cada transição entre esses níveis corresponde a um bit. Elas são ideais para transmissão de dados, pois permitem representar informações binárias de forma clara e eficiente, com a frequência da onda determinando a taxa de bits (baud rate).



Base 2 - Kilobytes

    A base 2 é um sistema numérico utilizado em computação, onde os números são representados usando apenas os dígitos 0 e 1 (bits). Cada dígito binário representa uma potência de 2.
    Conversão de Bits, Bytes e Kilobytes:


  • Bit (b): A menor unidade de dados, representando 0 ou 1.
  • Byte (B): Composto por 8 bits. Exemplo: 10101100 é um byte.
  • Kilobyte (KB): 1 KB = 1.024 bytes (não 1.000, como em SI).

    • Exemplos de conversão:


  • 1 Byte = 8 bits
  • 1 Kilobyte (KB) = 1.024 bytes = 8.192 bits

    • Essas unidades são fundamentais para armazenar e medir a quantidade de dados em sistemas computacionais.
